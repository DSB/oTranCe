Release Notes - oTranCe - Version oTranCe 1.0.0

Bug

  • [OTC-159] - Adding new language / user: Issues with pop-up on validation errors
  • [OTC-165] - When creating a new language the flag is not uploaded correctly if input was invalid
  • [OTC-166] - Wrong statistics for a language if somebody translates more than one language
  • [OTC-168] - "Hits" are not counted correctly with the filter "Search in translations:"
  • [OTC-169] - Search by value and file template
  • [OTC-170] - Home shows users that edited instead of users with edit rights
  • [OTC-176] - Paginator does not work correctly on admin pages if number of records per page is not set
  • [OTC-180] - It is possible to click the inline edit icon twice which adds more textareas to the same phrase.
  • [OTC-183] - Uploading a new flag for a language that is also present in the gui of OTC makes the icon in the upper right selectbox disappear.
  • [OTC-184] - If the project name is very long, the name overlaps the menue name in the right content area.
  • [OTC-185] - The export doesn't work if a prefix for the db tables is used.
  • [OTC-186] - When importing the wrong importer ist used at step analyze.

Improvement

  • [OTC-161] - Check for selected language
  • [OTC-164] - Do not allow user to skip last setup step
  • [OTC-181] - At registration the user is able to add javascript to his username and/or other fields. A filter to strip such things should be added.

Story

  • [OTC-86] - As an admin I want to enable/disable which import methods are available for the project.
  • [OTC-118] - Duplicate file templates
  • [OTC-162] - Validation messages when saving file templates
  • [OTC-167] - As an administrator, I want to see the statistics of a user straight away from user administration
  • [OTC-173] - As a new user I want to be informed if an admin activates my account.
  • [OTC-174] - As a translator I want to be informed if an admin grants new edit rights to my user profile.
  • [OTC-175] - Request language edit rights as registered user
  • [OTC-177] - If I get e-mails from the oTranCe system I want to get them in the interface language I've set in my user profile.
  • [OTC-182] - Register/Approved account should be listed in the log
  • [OTC-187] - Error page should be improved
